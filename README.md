# ⛅ 날씨어때  백엔드

### 🙌소개
- 2025-1학기 시스템종합설계(캡스톤디자인) (21003758-001) 프로젝트
- <오늘날씨어때> 팀의 <날씨어때> 프로젝트 백엔드 레포지토리입니다.

### 🚀 주요 기능
- 회원가입 및 로그인 (JWT 기반 인증)
- 체감 온도 및 착장 의류 기록
- 사용자 체질 관리 및 옷장 관리
- 데이터 암호화 및 개인정보 보호 기능 (AES)

### 🛠 기술 스택
- Java 17, Spring Boot 3.4.4  
- PostgreSQL 17  
- Redis (JWT 토큰 블랙리스트 처리)  
- Docker, AWS EC2  
- GitHub Actions (CI/CD)
